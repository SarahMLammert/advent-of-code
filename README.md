# Advent of Code Solutions (Python)

This repository contains my Python solutions to Advent of Code challenges, with an emphasis on correctness, clarity, and debugging.

## Overview

Advent of Code is an annual series of programming challenges that test problem-solving, algorithmic thinking, and attention to detail.  
I use these problems to practice writing clean, reliable Python code and reasoning through edge cases and performance trade-offs.

## Repository Structure

Solutions are organized by year and day.

Example structure:
- `2015/`
  - `day1.py`
  - `day2.py`
  - ...

Each script solves a single day's challenge.

## Running the Code

Each solution is a Python script that reads input from a text file in the same directory.

Example:
day1.py

## Approach

When solving these problems, I focus on:

    Writing clear and readable Python

    Validating correctness using provided examples

    Debugging edge cases and unexpected behavior

    Improving efficiency where appropriate without sacrificing clarity

## What I’ve Learned

Through Advent of Code, I’ve strengthened my ability to:

    Break down ambiguous problem statements

    Reason about data structures and algorithms

    Debug non-obvious logical errors

    Write maintainable Python under time constraints
